# МГТС Бизнес - Новый сайт

Современный корпоративный сайт для МГТС Бизнес, разработанный на основе технического задания.

## Структура проекта

```
SiteMGTS/
├── css/
│   └── style.css          # Основные стили (дизайн-система)
├── js/
│   └── main.js            # JavaScript функциональность
├── images/                # Изображения и ресурсы
├── index.html             # Главная страница
├── business/              # Раздел "Бизнес"
│   ├── index.html         # Hub-страница для бизнеса
│   └── internet/          # Категория "Интернет"
│       ├── index.html     # Страница категории
│       └── gpon/          # Услуга "GPON"
│           └── index.html # Страница услуги
└── README.md              # Этот файл
```

## Особенности

### Дизайн-система
- Современный минималистичный дизайн
- Адаптивная верстка (mobile-first)
- Цветовая палитра МГТС/МТС
- Компонентный подход

### Функциональность
- Мега-меню для навигации
- Breadcrumbs на всех страницах
- Формы с валидацией
- Плавная прокрутка
- Lazy loading изображений
- Анимации при прокрутке

### SEO
- Семантическая разметка HTML5
- Schema.org разметка (Organization, Service, BreadcrumbList)
- Оптимизированные мета-теги
- Правильная структура заголовков

### Производительность
- Минималистичный код
- Оптимизированные стили
- Lazy loading
- Быстрая загрузка страниц

## Как использовать

1. Откройте `index.html` в браузере
2. Все страницы используют относительные пути
3. Для полной функциональности нужен веб-сервер

## Кросс-платформенность

Проект полностью кросс-платформенный и работает на:
- ✅ **Windows** (10/11)
- ✅ **macOS** (10.14+)
- ✅ **Linux** (Ubuntu, Debian, Fedora и др.)

### Python скрипты

Все Python скрипты работают на всех платформах:

```bash
# Копирование шрифтов МТС
python copy_mts_fonts.py

# Обновление навигации
python update_nav.py

# Сбор структуры сайта
python site_structure_builder.py
```

**Примечание:** На macOS/Linux используйте `python3` вместо `python` если необходимо.

### Системные требования

- Python 3.7+
- Chrome/Chromium браузер (для Selenium скриптов)
- Доступ к интернету (для загрузки зависимостей)

## Следующие шаги

Для полноценного запуска сайта рекомендуется:

1. Добавить изображения в папку `images/`
2. Настроить бэкенд для обработки форм
3. Добавить остальные страницы (операторы, застройщики, партнеры, госсектор)
4. Интегрировать с CMS (если требуется)
5. Настроить аналитику (Google Analytics, Яндекс.Метрика)
6. Оптимизировать изображения
7. Настроить SEO (sitemap.xml, robots.txt)
