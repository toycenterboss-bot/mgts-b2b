# Изображения для сайта МГТС Бизнес

## Необходимые изображения

### Логотип
- `logo-mgts.svg` - Логотип МГТС (SVG формат)
- `logo-mgts.png` - Логотип МГТС (PNG, fallback)
- `favicon.ico` - Иконка сайта

### Иконки услуг (можно использовать эмодзи или иконки)
- Интернет: 🌐 или кастомная иконка
- Телефония: 📞 или кастомная иконка
- Облако: ☁️ или кастомная иконка
- Безопасность: 🔒 или кастомная иконка
- ТВ: 📺 или кастомная иконка
- Умный офис: 🏢 или кастомная иконка

### Фоновые изображения
- Hero background (опционально) - можно использовать градиенты из CSS

### Изображения для кейсов
- Логотипы компаний-клиентов
- Фото офисов/объектов (если есть)

## Рекомендации

1. **Формат**: Используйте современные форматы (WebP) с fallback на PNG/JPG
2. **Размеры**: Оптимизируйте изображения для веб-использования
3. **Responsive**: Используйте srcset для адаптивных изображений
4. **Lazy loading**: Все изображения должны иметь атрибут loading="lazy"

## Временное решение

На данный момент сайт использует эмодзи вместо изображений. Для production версии рекомендуется заменить на:
- SVG иконки из библиотеки (например, Font Awesome, Material Icons)
- Фотографии услуг и кейсов
- Логотип компании
