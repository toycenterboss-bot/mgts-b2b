# Компоненты сайта

## Описание

Эта папка содержит переиспользуемые компоненты сайта:
- `header.html` - Шапка сайта с навигацией и mega-menu
- `footer.html` - Подвал сайта со ссылками
- `sidebar-about.html` - Боковое меню для раздела "О компании"

## Использование

### Автоматическая загрузка через JavaScript

1. Добавьте плейсхолдер на странице:
```html
<div data-component="header"></div>
<div data-component="footer"></div>
<div data-component="sidebar-about"></div>
```

2. Подключите скрипт загрузки компонентов:
```html
<script src="js/components-loader.js"></script>
```

3. Компоненты автоматически загрузятся при загрузке страницы.

### Ручная загрузка через JavaScript

```javascript
// Загрузка компонента
ComponentLoader.load('header', '[data-component="header"]');

// Загрузка с опциями
ComponentLoader.load('footer', '[data-component="footer"]', {
    basePath: '../',  // Базовый путь (автоматически определяется)
    onLoad: function(target) {
        // Колбэк после загрузки
        console.log('Footer loaded');
    }
});
```

### Загрузка через iframe (альтернативный метод)

```html
<iframe src="components/header.html" style="border: none; width: 100%;"></iframe>
```

**Примечание:** Использование iframe не рекомендуется из-за проблем с SEO и навигацией.

## Обновление путей

Компоненты автоматически обновляют относительные пути в зависимости от уровня вложенности страницы:
- Корень: пути без префикса
- Уровень 1: `../`
- Уровень 2: `../../`
- И т.д.

Все ссылки и изображения с атрибутом `data-base-path` автоматически обновляются.

## Структура компонентов

### header.html
Содержит:
- Логотип и навигацию
- Mega-menu для "Услуги"
- Mega-menu для "Сегменты"
- Mega-menu для "О компании"

### footer.html
Содержит:
- Ссылки на услуги
- Ссылки на сегменты
- Ссылки на разделы "О компании"
- Контактную информацию
- Копирайт

### sidebar-about.html
Содержит:
- Боковое меню для раздела "О компании"
- Подменю для "Деловая этика и комплаенс"
- Подменю для "Корпоративное управление"
- JavaScript для управления подменю

## Обновление компонентов

При изменении компонента все страницы автоматически получат обновления при следующей загрузке.

## Миграция существующих страниц

Для миграции существующей страницы:

1. Замените блок `<header>...</header>` на `<div data-component="header"></div>`
2. Замените блок `<footer>...</footer>` на `<div data-component="footer"></div>`
3. Добавьте `<script src="js/components-loader.js"></script>` перед закрывающим тегом `</body>`
4. Убедитесь, что пути к CSS и другим ресурсам корректны
