⚠️ Внимание!

Вы открыли страницу через file:// протокол. Для работы компонентов необходимо использовать локальный HTTP сервер.

Запустите один из вариантов:
• python -m http.server 8000
• Или используйте START_SERVER.bat (Windows)

Затем откройте: http://localhost:8000/test-components.html
