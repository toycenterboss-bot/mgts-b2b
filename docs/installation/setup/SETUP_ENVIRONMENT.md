# Установка окружения для CMS (Этап 1)

## Статус проверки инструментов

❌ **Node.js** - не установлен
❌ **npm** - не установлен  
❌ **Git** - не установлен

## Инструкции по установке

### 1. Установка Node.js и npm

**Windows:**
1. Перейти на https://nodejs.org/
2. Скачать LTS версию (рекомендуется 18.x или выше)
3. Запустить установщик
4. Следовать инструкциям установщика
5. Убедиться, что опция "Add to PATH" отмечена
6. Перезапустить терминал/PowerShell

**Проверка установки:**
```powershell
node --version    # Должно показать v18.x.x или выше
npm --version     # Должно показать 9.x.x или выше
```

**Альтернатива через Chocolatey (если установлен):**
```powershell
choco install nodejs
```

### 2. Установка Git

**Windows:**
1. Перейти на https://git-scm.com/download/win
2. Скачать установщик
3. Запустить установщик
4. Следовать инструкциям (можно оставить настройки по умолчанию)
5. Перезапустить терминал/PowerShell

**Проверка установки:**
```powershell
git --version
```

**Альтернатива через Chocolatey:**
```powershell
choco install git
```

### 3. Установка PostgreSQL (опционально, для продакшена)

**Для разработки можно использовать SQLite (входит в Strapi)**

**Если нужен PostgreSQL для продакшена:**

**Windows:**
1. Перейти на https://www.postgresql.org/download/windows/
2. Скачать установщик
3. Запустить установщик
4. Запомнить пароль для пользователя postgres
5. Порт по умолчанию: 5432

**Проверка установки:**
```powershell
psql --version
```

## После установки всех инструментов

Выполните следующие команды для проверки:

```powershell
node --version
npm --version
git --version
```

Все команды должны вернуть версии без ошибок.

## Следующие шаги

После успешной установки всех инструментов:
1. Перезапустите терминал/PowerShell
2. Перейдите в директорию проекта
3. Продолжите с Шага 1.2: Создание нового Strapi проекта
