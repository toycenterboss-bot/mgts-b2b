# План унификации меню навигации

## Принятое решение: Вариант 1 - Единое меню везде

### Принципы:

1. **На всех страницах одинаковое главное меню:**
   - Главная
   - Услуги (с mega-menu на главной)
   - Сегменты (с mega-menu на главной)
   - Решения
   - Поддержка
   - О компании
   - Контакты
   - Телефон (всегда справа)

2. **Mega-menu работает только на главной странице**
   - На остальных страницах просто ссылки

3. **Активная страница выделяется автоматически** через JavaScript

4. **Breadcrumbs показывают путь** на всех вложенных страницах

## Структура меню:

```
Главная | Услуги | Сегменты | Решения | Поддержка | О компании | Контакты | 📞 8 800 250-0-250
```

## Преимущества:

✅ Единообразие - пользователь всегда видит полную навигацию
✅ Не теряется доступ к разделам
✅ Понятная структура
✅ Легко поддерживать
