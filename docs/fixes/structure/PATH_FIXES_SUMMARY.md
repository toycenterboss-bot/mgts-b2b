## Для следующих этапов

### Настройка переменных окружения

Перед запуском скриптов извлечения контента (Этап 2), установите:

**Windows PowerShell:**
```powershell
$env:SITE_ROOT = "C:\Users\abefremov\SiteMGTS"
$env:CMS_ROOT = "C:\Users\abefremov\mgts-cms"
```

**Windows CMD:**
```cmd
set SITE_ROOT=C:\Users\abefremov\SiteMGTS
set CMS_ROOT=C:\Users\abefremov\mgts-cms
```

**Linux/Mac:**
```bash
export SITE_ROOT=/path/to/SiteMGTS
export CMS_ROOT=/path/to/mgts-cms
```

### Создание файла .env в mgts-backend

Создайте файл `.env` в директории `mgts-backend`:

```env
SITE_ROOT=C:\Users\abefremov\SiteMGTS
CMS_ROOT=C:\Users\abefremov\mgts-cms
```
