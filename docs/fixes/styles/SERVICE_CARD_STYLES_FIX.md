# Исправление стилей для service-card элементов

## Проблема

На страницах раздела "О компании" элементы `service-card` отображаются с белым текстом, из-за чего контент не виден.

## Причина

1. **Глобальные стили для h3** - могут наследоваться от родительских элементов
2. **Отсутствие явных стилей** - элементы внутри `service-card` не имеют явно установленного цвета текста
3. **Конфликты стилей** - возможно, стили из других секций (например, `.hero h3`) применяются к элементам внутри `service-card`

## Решение

### 1. Добавлены явные стили в CSS

В файле `SiteMGTS/css/style.css` добавлены стили с `!important` для гарантированного применения:

```css
.service-card h3 {
  color: var(--color-gray-900) !important;
  font-weight: var(--font-weight-bold);
}

.service-card-body h3 {
  color: var(--color-gray-900) !important;
}

.service-card p {
  color: var(--color-gray-700) !important;
}

.service-card-body p {
  color: var(--color-gray-700) !important;
}

.service-card-body {
  color: var(--color-gray-900) !important;
}

.service-card-body * {
  color: inherit;
}

.service-card-icon {
  color: var(--color-white) !important; /* Иконка должна быть белой на цветном фоне */
}

.service-card-icon i {
  color: var(--color-white) !important; /* Font Awesome иконки должны быть белыми */
}
```

### 2. Проверка HTML в Strapi

Проверено, что в HTML из Strapi:
- ✅ Все div элементы имеют классы
- ✅ Нет inline стилей с `color: white`
- ✅ Структура соответствует правилам типизации

### 3. Рекомендации

Если проблема сохраняется после очистки кэша браузера:

1. **Проверить, не применяются ли стили из других секций:**
   - `.hero h3` - может наследоваться
   - `.section h3` - может наследоваться
   - Глобальные стили для `h3`

2. **Убедиться, что элементы имеют правильные классы:**
   - `.service-card` - основной класс карточки
   - `.service-card-body` - тело карточки
   - `.service-card-icon` - иконка карточки

3. **Проверить в DevTools:**
   - Открыть DevTools (F12)
   - Выбрать элемент с белым текстом
   - Проверить Computed styles
   - Найти, какой стиль устанавливает `color: white`
