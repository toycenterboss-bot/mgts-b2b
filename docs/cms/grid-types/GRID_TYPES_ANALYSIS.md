## Проблема

**Текущая логика в `enhanceServiceCard`:**
```javascript
if (link && !isCardLink) {
    // Создать кнопку "Узнать больше"
}
```

**Проблема:** `link` по умолчанию равен `'#'` (строка 404), поэтому кнопка создается даже для карточек без ссылок.

---

## Решение

Нужно изменить логику определения наличия ссылки:

1. **Для Navigation Cards:** Проверить, является ли карточка `<a>` тегом (`isCardLink === true`)
   - ❌ Не добавлять кнопку "Узнать больше"

2. **Для Info Cards:** Проверить, что нет ссылок вообще (`link === '#'` или `link === ''` или нет ссылок внутри)
   - ❌ Не добавлять кнопку "Узнать больше"

3. **Для Service Cards with Links:** Проверить, что есть ссылка внутри карточки (`link !== '#'` и `link !== ''` и `!isCardLink`)
   - ✅ Добавить кнопку "Узнать больше"

4. **Для Service Cards without Links:** Проверить, что нет ссылок (`link === '#'` или `link === ''`)
   - ❌ Не добавлять кнопку "Узнать больше"

---

## План исправления

1. ✅ Изменить логику определения ссылки в `enhanceServiceCard`:
   - Если карточка является `<a>` тегом → `link = null` (не создавать кнопку)
   - Если ссылка внутри карточки → `link = href` (создать кнопку)
   - Если ссылки нет → `link = null` (не создавать кнопку)

2. ✅ Изменить условие создания кнопки:
   ```javascript
   if (link && link !== '#' && link !== '' && !isCardLink) {
       // Создать кнопку "Узнать больше"
   }
   ```

3. ✅ Протестировать на всех страницах раздела "О компании"
