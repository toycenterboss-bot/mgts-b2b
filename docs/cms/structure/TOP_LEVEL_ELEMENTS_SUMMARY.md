## Примененные исправления

### 1. Добавление класса `section-title`

**Правило:**
- Все заголовки секций (`<h2>`) должны иметь класс `section-title`
- Исключения: заголовки в формах заказа, с специальными классами, внутри карточек

**Примеры исправлений:**
```html
<!-- До исправления -->
<h2>Заголовок секции</h2>

<!-- После исправления -->
<h2 class="section-title">Заголовок секции</h2>
```
