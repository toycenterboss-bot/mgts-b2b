## 🔍 Ручное тестирование

### Тест 1: Создание страницы через админ-панель

1. Откройте http://localhost:1337/admin
2. Content Manager → Pages → Create new entry
3. Заполните:
   - Slug: `test-page`
   - Title: `Тестовая страница`
   - Content: `<p>Тестовый контент</p>`
4. Save → Publish
5. Проверьте через API: `curl http://localhost:1337/api/pages/slug/test-page`

### Тест 2: Загрузка изображения

1. Media Library → Add new assets
2. Выберите изображение
3. Дождитесь загрузки
4. Проверьте URL изображения в браузере

### Тест 3: Загрузка контента из API на фронтенде

1. Откройте http://localhost:8001/index.html?cms=true
2. Откройте консоль браузера (F12)
3. Проверьте сообщения:
   - `[CMS Loader] Loading page from API: index`
   - `[CMS Loader] Page loaded successfully: index`
4. Проверьте, что контент отображается корректно
