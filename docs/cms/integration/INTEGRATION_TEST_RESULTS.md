## 🔍 Как протестировать вручную

### 1. Тест загрузки контента из API

**Откройте в браузере:**
```
http://localhost:8001/index.html?cms=true
```

**Проверьте консоль браузера (F12):**
- Должны быть сообщения:
  - `[CMS Loader] Loading page from API: index`
  - `[CMS Loader] Page loaded successfully: index`

**Проверьте страницу:**
- Контент должен загрузиться из Strapi API
- Метаданные должны обновиться
- Hero секция должна отображаться

### 2. Тест без параметра cms=true

**Откройте:**
```
http://localhost:8001/index.html
```

**Ожидаемое поведение:**
- На localhost API должен включиться автоматически
- Или будет использован статический контент

### 3. Тест API напрямую

```bash
# Получить страницу
curl http://localhost:1337/api/pages/slug/index

# Получить все страницы
curl http://localhost:1337/api/pages?pagination[pageSize]=5

# Получить меню
curl http://localhost:1337/api/menu
```
