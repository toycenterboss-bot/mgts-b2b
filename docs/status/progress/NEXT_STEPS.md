## 📝 Детальный план действий

### 1. Создать API клиент

**Файл:** `SiteMGTS/js/api-client.js`

```javascript
class StrapiAPI {
  constructor(baseURL = 'http://localhost:1337') {
    this.baseURL = baseURL;
  }
  
  async getPage(slug) {
    const response = await fetch(
      `${this.baseURL}/api/pages?filters[slug][$eq]=${slug}&populate=*`
    );
    const data = await response.json();
    return data.data[0];
  }
  
  async getAllPages() {
    const response = await fetch(`${this.baseURL}/api/pages?populate=*`);
    const data = await response.json();
    return data.data;
  }
}
```
