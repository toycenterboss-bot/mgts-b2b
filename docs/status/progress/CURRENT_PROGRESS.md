## 📋 Следующие действия

### 1. Создать API токен в Strapi

1. Откройте http://localhost:1337/admin
2. Перейдите в **Settings** → **API Tokens**
3. Нажмите **Create new API Token**
4. Заполните:
   - **Name:** `Import Script`
   - **Token duration:** `Unlimited`
   - **Token type:** `Full access`
5. Скопируйте токен

### 2. Запустить импорт

```bash
cd /Users/andrey_efremov/Downloads/runs/mgts-backend
export STRAPI_API_TOKEN="your_token_here"
node scripts/migration/import-content.js
```

### 3. Проверить импортированный контент

После импорта:
- Проверить в админ-панели: Content Manager → Pages
- Проверить через API: `curl http://localhost:1337/api/pages`
