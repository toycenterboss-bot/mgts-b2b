## Шаг 1.2: Создание нового Strapi проекта

**Статус:** ⏸️ Ожидает установки Node.js и npm

**Команды для выполнения после установки:**
```powershell
# Перейти в родительскую директорию
cd ..

# Создать директорию для CMS
mkdir mgts-cms
cd mgts-cms

# Установить Strapi CLI глобально
npm install -g @strapi/strapi

# Создать новый проект
npx create-strapi-app@latest mgts-backend --quickstart
```

## Блокирующие факторы

1. **Node.js не установлен** - необходимо установить перед продолжением
2. **npm не установлен** - устанавливается вместе с Node.js
3. **Git не установлен** - необходимо для версионирования (опционально)

## Рекомендации

1. Установите Node.js LTS версию с https://nodejs.org/
2. Установите Git с https://git-scm.com/download/win
3. Перезапустите терминал после установки
4. Проверьте установку командой `node --version`
5. Продолжите с Шага 1.2

## Следующие действия

После установки всех инструментов выполните:
```powershell
# Проверка
node --version
npm --version
git --version

# Если все работает, продолжите с создания проекта
cd ..
mkdir mgts-cms
cd mgts-cms
npm install -g @strapi/strapi
npx create-strapi-app@latest mgts-backend --quickstart
```
