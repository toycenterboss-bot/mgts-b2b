# Обновление шрифтов на корпоративные МТС/МГТС - Завершено ✅

## Выполненные изменения:

### Обновление типографики:

Согласно корпоративным стандартам МТС/МГТС, обновлены шрифты в CSS файле:

**Было:**
- `--font-family-primary: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;`
- `--font-family-heading: "Inter", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;`

**Стало:**
- `--font-family-primary: Arial, "Helvetica Neue", Helvetica, sans-serif;`
- `--font-family-heading: Arial, "Helvetica Neue", Helvetica, sans-serif;`

### Обоснование:

Согласно руководству по стилю МТС:
- **MTS Text Regular** и **MTS Compact Regular** - корпоративные шрифты для основного текста
- **MTS Sans** - корпоративный шрифт для заголовков
- **Arial** - стандартная замена, используемая когда корпоративные шрифты MTS недоступны
  - Arial Regular - для основного текста (замена MTS Text)
  - Arial Bold - для заголовков (замена MTS Sans Bold)

Arial доступен на всех основных операционных системах (Windows, macOS, Linux) и обеспечивает:
- Высокую читаемость
- Соответствие корпоративному стилю МТС/МГТС
- Кросс-платформенную совместимость
- Быструю загрузку (не требует загрузки внешних шрифтов)

### Результат:

Теперь все страницы сайта используют шрифты, соответствующие корпоративному стилю МТС/МГТС, аналогично сайту https://business.mgts.ru/.
