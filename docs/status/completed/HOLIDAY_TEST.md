# Как проверить новогоднюю анимацию

Чтобы проверить, что новогодняя анимация работает:

1. **Измените дату в браузере** (для тестирования):
   - Откройте DevTools (F12)
   - Перейдите во вкладку "Console"
   - Выполните:
   ```javascript
   // Временно изменить дату для теста
   const originalDate = Date;
   Date = class extends originalDate {
     constructor(...args) {
       if (args.length === 0) {
         super(new originalDate(2024, 11, 25)); // 25 декабря 2024
       } else {
         super(...args);
       }
     }
   };
   // Перезагрузите страницу
   location.reload();
   ```

2. **Или просто дождитесь 24 декабря - 8 января**, когда анимация активируется автоматически.

3. **Что вы должны увидеть:**
   - Голубой градиент на hero-секциях вместо обычного синего
   - Падающие снежинки ❄ по всему экрану
   - Плавную анимацию снегопада

4. **Проверка через консоль:**
   ```javascript
   // Проверить классы на body
   document.body.classList.contains('holiday-newyear') // должно быть true
   
   // Проверить наличие снегопада
   document.querySelector('.snow-container') // не должно быть null
   ```
