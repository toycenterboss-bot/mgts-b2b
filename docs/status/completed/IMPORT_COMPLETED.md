### ✅ Автоматический импорт при запуске Strapi

**Реализация:**
- Добавлен код импорта в `src/index.ts` (функция `bootstrap`)
- Импорт запускается автоматически при первом запуске Strapi
- Проверяется наличие страниц - если их нет, запускается импорт

**Процесс:**
1. При запуске Strapi проверяется количество страниц
2. Если страниц нет (0), запускается импорт
3. Читается файл `parsed-content.json`
4. Для каждой страницы:
   - Адаптация данных под схему Page
   - Проверка существования по slug
   - Создание или обновление страницы

## 🔍 Проверка импортированного контента

### Через API:
```bash
# Получить все страницы
curl http://localhost:1337/api/pages

# Получить страницу по slug
curl "http://localhost:1337/api/pages?filters[slug][$eq]=index"

# Получить количество страниц
curl "http://localhost:1337/api/pages?pagination[pageSize]=1"
```

### Через админ-панель:
1. Откройте http://localhost:1337/admin
2. Content Manager → Pages
3. Должно быть 41 страница

---

## 📝 Структура импортированных данных

Каждая страница содержит:
- **slug** - уникальный идентификатор
- **title** - заголовок страницы
- **metaDescription** - описание для SEO
- **metaKeywords** - ключевые слова
- **heroTitle** - заголовок Hero секции
- **heroSubtitle** - подзаголовок Hero секции
- **content** - основной контент (Rich Text HTML)
- **breadcrumbs** - навигационные крошки (JSON)
- **sidebar** - тип бокового меню (none/about)

## ⚙️ Настройки

**Автоматический импорт:**
- Включен в `src/index.ts`
- Запускается только если страниц нет (0)
- Можно отключить, закомментировав код в `bootstrap`

**Ручной импорт:**
Если нужно импортировать вручную:
1. Удалите все страницы через админ-панель
2. Перезапустите Strapi
3. Импорт запустится автоматически
