### Шаг 1.1: Установка инструментов

**Требуется установить:**
1. **Node.js** (обязательно)
   - Скачать с https://nodejs.org/
   - Установить LTS версию (18.x или выше)
   - Перезапустить терминал

2. **Git** (рекомендуется)
   - Скачать с https://git-scm.com/download/win
   - Установить с настройками по умолчанию
   - Перезапустить терминал

**После установки:**
```powershell
# Проверить установку
.\check_environment.ps1
```

### Шаг 1.2: Создание проекта Strapi

**После успешной установки инструментов:**

```powershell
# Вариант 1: Использовать скрипт
.\setup_step1.ps1

# Вариант 2: Выполнить вручную
cd ..
mkdir mgts-cms
cd mgts-cms
npm install -g @strapi/strapi
npx create-strapi-app@latest mgts-backend --quickstart
```

### Шаг 1.3: Настройка базы данных

**Для разработки:** SQLite (включен в quickstart)
**Для продакшена:** PostgreSQL (требует установки)
