### 1. Дождаться запуска Strapi

Strapi запускается в фоновом режиме. Подождите 30-60 секунд, затем:

```bash
# Проверить статус
curl http://localhost:1337/admin

# Или открыть в браузере
open http://localhost:1337/admin
```

### 2. Настроить переменные окружения постоянно (опционально)

```bash
# Добавить в ~/.zshrc
echo 'export SITE_ROOT="/Users/andrey_efremov/Downloads/runs/SiteMGTS"' >> ~/.zshrc
source ~/.zshrc
```

### 3. Запустить локальный сервер для фронтенда

В **новом терминале**:

```bash
cd /Users/andrey_efremov/Downloads/runs/SiteMGTS
./START_SERVER.sh
```

Или:
```bash
python3 -m http.server 8000
```

Откроется сайт: http://localhost:8000
