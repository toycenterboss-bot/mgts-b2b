## 🚀 Использование

### Включить загрузку из API:

**Вариант 1:** Добавить параметр в URL
```
http://localhost:8001/index.html?cms=true
```

**Вариант 2:** Установить в localStorage
```javascript
localStorage.setItem('useCMS', 'true');
```

**Вариант 3:** Автоматически на localhost
- На `localhost` или `127.0.0.1` API включается автоматически

### Использовать шаблон для новых страниц:

1. Скопировать `page-template.html`
2. Изменить slug в функции `getSlugFromURL()` или использовать из URL
3. Страница автоматически загрузит контент из Strapi

---

## 🔍 Как это работает

1. **При загрузке страницы:**
   - `cms-loader.js` проверяет, нужно ли использовать API
   - Если да, извлекает slug из URL
   - Загружает данные страницы через `StrapiAPI.getPage(slug)`

2. **При успешной загрузке:**
   - Обновляет метаданные (title, description, keywords)
   - Рендерит Hero секцию
   - Вставляет контент в основную область
   - Рендерит breadcrumbs
   - Загружает sidebar если нужно

3. **При ошибке:**
   - Логирует ошибку в консоль
   - Использует статический контент (fallback)
   - Показывает сообщение об ошибке (в шаблоне)
