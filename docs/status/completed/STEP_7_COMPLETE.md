## 🎯 Основные функции для редакторов

### Создание страниц

1. Content Manager → Pages → Create new entry
2. Заполнить обязательные поля:
   - Slug (уникальный идентификатор)
   - Title (заголовок)
3. Заполнить опциональные поля
4. Save → Publish

### Работа с изображениями

1. Media Library → Add new assets
2. Выбрать файл
3. Изображение сохранится локально
4. URL: `http://localhost:1337/uploads/...`

### Редактирование контента

- Rich Text редактор для форматирования
- Поддержка HTML
- Вставка изображений из Media Library
