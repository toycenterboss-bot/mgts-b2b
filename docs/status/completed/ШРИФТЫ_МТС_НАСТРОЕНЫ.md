# Шрифты МТС настроены ✅

## Что сделано:

1. ✅ **Найдены корпоративные шрифты МТС** в системе Windows (`C:\Windows\Fonts`)
2. ✅ **Добавлены @font-face правила** в `css/style.css` для подключения шрифтов МТС:
   - MTS Text (Regular, Medium, Bold) - для основного текста
   - MTS Sans (Regular, Medium, Bold) - для заголовков
3. ✅ **Обновлены CSS переменные**:
   - `--font-family-primary: "MTS Text", Arial, ...` - для основного текста
   - `--font-family-heading: "MTS Sans", Arial, ...` - для заголовков

## Что нужно сделать:

### Вариант 1: Автоматическое копирование (рекомендуется)

Запустите один из скриптов:

**Windows Batch файл:**
```
copy_mts_fonts.bat
```

**PowerShell скрипт:**
```powershell
.\copy_mts_fonts.ps1
```

### Вариант 2: Ручное копирование

1. Создайте папку `fonts` в папке `SiteMGTS` (если её нет)
2. Скопируйте следующие файлы из `C:\Windows\Fonts` в `SiteMGTS\fonts`:
   - MTSText-Regular.otf
   - MTSText-Medium.otf
   - MTSText-Bold.otf
   - MTSSans-Regular.otf
   - MTSSans-Medium.otf
   - MTSSans-Bold.otf

## После копирования:

После того как файлы шрифтов будут скопированы в папку `SiteMGTS\fonts`, они автоматически будут использоваться на всех страницах сайта!

## Используемые шрифты:

- **MTS Text** - основной корпоративный шрифт МТС для текста
- **MTS Sans** - корпоративный шрифт МТС для заголовков

Эти шрифты соответствуют корпоративным стандартам МТС/МГТС и используются на сайте https://business.mgts.ru/
