# Инструкция по копированию шрифтов МТС

## Автоматическое копирование

Запустите PowerShell скрипт из папки `SiteMGTS`:

```powershell
.\copy_mts_fonts.ps1
```

## Ручное копирование

Если скрипт не работает, скопируйте следующие файлы из `C:\Windows\Fonts` в папку `SiteMGTS\fonts`:

1. MTSText-Regular.otf
2. MTSText-Medium.otf
3. MTSText-Bold.otf
4. MTSSans-Regular.otf
5. MTSSans-Medium.otf
6. MTSSans-Bold.otf

### Создание папки fonts

Если папка `fonts` не существует, создайте её в папке `SiteMGTS`.

### После копирования

CSS файл уже обновлен с правильными @font-face правилами. После копирования шрифтов они автоматически будут использоваться на сайте.
