# Инструкция по полному извлечению контента страницы "О компании"

## Запуск скрипта

Запустите скрипт из корневой директории проекта (где находится файл `about_original.html`):

```bash
python extract_about_full.py
```

Или из папки SiteMGTS (если скрипт там):

```bash
cd SiteMGTS
python extract_about_full.py
```

## Что делает скрипт

1. **Извлекает левое меню** - находит все пункты меню слева на странице
2. **Извлекает интерактивную историю** - кликает по каждому периоду лет и получает соответствующий текст
3. **Извлекает все разделы** - находит все заголовки и связанный с ними контент

## Результат

Скрипт создаст файл `about_content_full.json` с полной структурой:
- `menu_items` - список пунктов меню
- `sections` - все разделы страницы
- `history_periods` - история по периодам с соответствующим контентом

## Важно

- Скрипт использует Selenium для интерактивного взаимодействия со страницей
- Некоторые периоды истории требуют клика для отображения контента
- Скрипт автоматически кликает по каждому периоду и извлекает текст
