## Как открыть сайт

### Вариант 1: Прямое открытие
Просто откройте файл `index.html` в браузере.

⚠️ **Примечание**: При прямом открытии некоторые функции могут не работать из-за ограничений безопасности браузера (CORS).

### Вариант 2: Локальный сервер (рекомендуется)

#### Python 3
```bash
cd SiteMGTS
python -m http.server 8000
```
Откройте в браузере: `http://localhost:8000`

#### Node.js (http-server)
```bash
npm install -g http-server
cd SiteMGTS
http-server
```

#### PHP
```bash
cd SiteMGTS
php -S localhost:8000
```

## Структура URL

После развертывания на сервере структура будет следующей:

```
/
├── / (главная)
├── /business/ (hub-страница бизнеса)
│   ├── /business/internet/ (категория интернет)
│   │   ├── /business/internet/gpon/ (услуга GPON)
│   │   ├── /business/internet/dedicated/ (услуга выделенный интернет)
│   │   └── /business/internet/office/ (услуга офисный интернет)
│   ├── /business/telephony/ (категория телефония)
│   ├── /business/cloud/ (категория облако)
│   └── /business/security/ (категория безопасность)
├── /operators/ (hub-страница операторов)
├── /developers/ (hub-страница застройщиков)
├── /partners/ (hub-страница партнеров)

```

## Поддержка браузеров

Сайт поддерживает современные браузеры:
- Chrome (последние 2 версии)
- Firefox (последние 2 версии)
- Safari (последние 2 версии)
- Edge (последние 2 версии)

Для старых браузеров может потребоваться полифиллы.

## Производительность

Целевые метрики:
- LCP (Largest Contentful Paint): < 2.5s
- FID (First Input Delay): < 100ms
- CLS (Cumulative Layout Shift): < 0.1
- PageSpeed Insights: > 90 (mobile), > 95 (desktop)
