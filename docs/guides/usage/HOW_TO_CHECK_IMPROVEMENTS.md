## 🔍 Как сравнить "До" и "После"

### Метод 1: Визуальное сравнение
1. Откройте главную страницу
2. Обратите внимание на:
   - Размер заголовков в Hero
   - Внешний вид кнопок
   - Работу слайдера "Почему МГТС?"
   - Навигацию с клавиатуры (Tab)

### Метод 2: Проверка через DevTools
1. Откройте DevTools (F12)
2. Проверьте стили элементов:
   - Hero секция: должен быть `background: linear-gradient`
   - Кнопки: должны иметь `box-shadow` и `min-height: 60px`
   - Слайдер: должны быть элементы `.mirror-slider__dots` и `.mirror-slider__arrow`

### Метод 3: Проверка функциональности
1. **Hero секция:** Заголовок должен быть крупным и читаемым
2. **CTA кнопки:** При наведении должны подниматься вверх
3. **Mirror Slider:** Должен автоматически переключаться
4. **Мобильное меню:** Иконка должна быть SVG, не эмодзи
5. **Клавиатурная навигация:** Tab должен показывать focus outline

## 💡 Советы по проверке

1. **Откройте страницу в разных браузерах:**
   - Chrome/Edge
   - Firefox
   - Safari

2. **Проверьте на разных устройствах:**
   - Десктоп (> 1024px)
   - Планшет (768px - 1024px)
   - Мобильный (< 768px)

3. **Проверьте интерактивность:**
   - Наведите курсор на кнопки
   - Кликните на элементы слайдера
   - Попробуйте навигацию с клавиатуры (Tab)

4. **Проверьте консоль браузера:**
   - Не должно быть ошибок JavaScript
   - Не должно быть ошибок загрузки ресурсов
