# 📖 Руководство для редакторов контента

## 🎯 Введение

Это руководство поможет вам работать с системой управления контентом (CMS) для сайта МГТС.

**Админ-панель:** http://localhost:1337/admin

---

## 📝 Создание и редактирование страниц

### Создать новую страницу

1. Откройте админ-панель: http://localhost:1337/admin
2. В левом меню найдите **Content Manager**
3. Выберите **Pages**
4. Нажмите **Create new entry** (справа вверху)

### Заполнение полей страницы

#### Обязательные поля:

- **Slug** (Уникальный идентификатор)
  - Формат: только латиница, цифры, дефисы и слэши
  - Примеры: `index`, `about/ethics`, `business/internet`
  - ⚠️ Важно: slug должен быть уникальным

- **Title** (Заголовок страницы)
  - Отображается в `<title>` и в браузере
  - Пример: `МГТС Бизнес - Цифровые решения для вашего бизнеса`

#### Опциональные поля:

- **Meta Description** (Описание для поисковых систем)
  - Краткое описание страницы (150-160 символов)
  - Используется в результатах поиска

- **Meta Keywords** (Ключевые слова)
  - Через запятую: `интернет, телефония, МГТС`

- **Hero Title** (Заголовок Hero секции)
  - Главный заголовок на странице
  - Отображается в секции `.hero`

- **Hero Subtitle** (Подзаголовок Hero секции)
  - Подзаголовок под основным заголовком

- **Content** (Основной контент)
  - Rich Text редактор
  - Поддерживает HTML разметку
  - Можно вставлять заголовки, списки, ссылки

- **Breadcrumbs** (Хлебные крошки)
  - JSON формат
  - Пример:
    ```json
    [
      {"name": "Главная", "url": "/"},
      {"name": "О компании", "url": "/about"},
      {"name": "Деловая этика", "url": "/about/ethics"}
    ]
    ```

- **Sidebar** (Боковое меню)
  - Выберите: `none` или `about`
  - `about` - для страниц раздела "О компании"

### Сохранение и публикация

1. После заполнения полей нажмите **Save**
2. Для публикации страницы нажмите **Publish**
3. ⚠️ Неопубликованные страницы не отображаются на сайте

## 🖼️ Работа с изображениями

### Загрузить изображение

1. Перейдите в **Media Library** (в левом меню)
2. Нажмите **Add new assets**
3. Выберите файл(ы) с компьютера
4. Дождитесь загрузки
5. Изображение сохранится локально в `public/uploads/`

### Использовать изображение в контенте

1. В Rich Text редакторе нажмите на иконку изображения
2. Выберите изображение из Media Library
3. Или вставьте URL изображения

### URL изображений

После загрузки изображение будет доступно по адресу:
```
http://localhost:1337/uploads/2024/12/filename.jpg
```

## 📄 Типы контента

### Page (Страница)

Основной тип контента для страниц сайта.

**Поля:**
- slug, title, metaDescription, heroTitle, content и др.

**Использование:**
- Создание новых страниц
- Редактирование существующих страниц

## 🔍 Поиск и фильтрация

### Найти страницу

1. В **Content Manager** → **Pages**
2. Используйте поиск вверху
3. Или фильтруйте по полям

### Фильтрация

- По slug
- По title
- По дате создания/обновления
- По статусу публикации

## ⚙️ Настройки

### Права доступа

- **Администратор:** Полный доступ ко всем функциям
- **Редактор:** Может создавать и редактировать контент
- **Автор:** Может создавать, но не публиковать

### Настройка профиля

1. Нажмите на аватар в правом верхнем углу
2. Выберите **Profile**
3. Измените имя, email, пароль

## 💡 Советы и рекомендации

### Создание slug

- Используйте латиницу и дефисы
- Делайте slug понятным и читаемым
- Примеры хороших slug:
  - ✅ `about/ethics`
  - ✅ `business/internet`
  - ✅ `contact-us`
- Примеры плохих slug:
  - ❌ `about_ethics` (используйте дефис)
  - ❌ `О компании` (только латиница)

### Работа с контентом

- Используйте Rich Text редактор для форматирования
- Добавляйте заголовки (H2, H3) для структуры
- Используйте списки для перечислений
- Добавляйте ссылки на другие страницы

### SEO оптимизация

- Заполняйте Meta Description для каждой страницы
- Используйте ключевые слова в Meta Keywords
- Делайте Title уникальным и информативным
- Используйте заголовки H1, H2, H3 в контенте

### Работа с изображениями

- Оптимизируйте изображения перед загрузкой
- Используйте понятные имена файлов
- Добавляйте альтернативный текст (alt) для изображений
- Не загружайте слишком большие файлы (>5MB)

## ❓ Часто задаваемые вопросы

### Как создать новую страницу?

1. Content Manager → Pages → Create new entry
2. Заполните slug и title (обязательно)
3. Заполните остальные поля
4. Save → Publish

### Как изменить существующую страницу?

1. Content Manager → Pages
2. Найдите нужную страницу
3. Нажмите на неё
4. Внесите изменения
5. Save → Publish

### Как удалить страницу?

1. Откройте страницу
2. Нажмите на три точки (⋮) в правом верхнем углу
3. Выберите **Delete**
4. Подтвердите удаление

### Где хранятся изображения?

Изображения хранятся локально в `public/uploads/` и доступны через URL:
```
http://localhost:1337/uploads/2024/12/filename.jpg
```

### Как изменить порядок страниц?

Страницы отображаются в алфавитном порядке по slug. Для изменения порядка можно использовать префиксы в slug (например: `01-about`, `02-services`).
