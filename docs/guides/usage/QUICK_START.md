## Если скрипт все еще не работает

Выполните команды вручную:

### Шаг 1: Проверка окружения
```powershell
.\check_environment.ps1
```

### Шаг 2: Создание структуры
```powershell
# Перейти в родительскую директорию
cd ..

# Создать директорию для CMS
mkdir mgts-cms
cd mgts-cms
```

### Шаг 3: Установка Strapi CLI
```powershell
npm install -g @strapi/strapi
```

### Шаг 4: Создание проекта
```powershell
npx create-strapi-app@latest mgts-backend --quickstart
```

## Что дальше

После создания проекта:
1. Перейдите в директорию проекта: `cd mgts-backend`
2. Запустите сервер: `npm run develop`
3. Откройте http://localhost:1337/admin
4. Создайте администратора
