# Быстрое решение проблемы с установкой Strapi

## Проблема
```
Эта программа заблокирована групповой политикой.
```

## Быстрое решение (2 минуты)

### Шаг 1: Установить create-strapi-app глобально

```powershell
npm install -g create-strapi-app
```

### Шаг 2: Создать проект

```powershell
cd C:\mgts-cms
create-strapi-app mgts-backend --quickstart
```

**Готово!** Это должно обойти ограничение npx.

---

## Если не помогло

### Альтернатива: Использовать скрипт ручной установки

```powershell
cd C:\mgts-cms
.\install_strapi_manual.ps1
```

Этот скрипт установит Strapi вручную без использования npx.

---

## Если ничего не работает

Обратитесь к системному администратору с запросом разрешить выполнение npm скриптов.
