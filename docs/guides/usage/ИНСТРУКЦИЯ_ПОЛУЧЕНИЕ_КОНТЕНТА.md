# Инструкция по получению содержимого страницы "О компании"

## Автоматический способ

Для получения содержимого с оригинального сайта запустите скрипт:

```bash
python fetch_about_mgts.py
```

Скрипт создаст два файла:
- `about_original.html` - полный HTML страницы
- `about_content.json` - извлеченное содержимое разделов в JSON формате

После этого содержимое можно будет перенести на новую страницу.

## Альтернативный способ

Если скрипт не работает, вы можете:
1. Открыть https://business.mgts.ru/about_mgts в браузере
2. Сохранить страницу как HTML (Ctrl+S)
3. Предоставить сохраненный HTML файл для анализа
