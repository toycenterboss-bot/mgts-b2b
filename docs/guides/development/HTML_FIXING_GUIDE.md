## План действий

### Шаг 1: Анализ страницы

```bash
node mgts-backend/scripts/analyze-page-structure.js business/telephony
```

Это покажет:
- Какие секции есть на странице
- Какие проблемы обнаружены
- Соответствие типизации

### Шаг 2: Исправление страницы

#### Вариант A: Автоматическое исправление (рекомендуется)

```bash
# Проверка (без сохранения)
node mgts-backend/scripts/fix-page-html.js business/telephony --dry-run

# Применение изменений
node mgts-backend/scripts/fix-page-html.js business/telephony
```

#### Вариант B: Ручное исправление в Strapi

1. Открыть страницу `business/telephony` в Strapi
2. Скопировать правильную структуру из `FIXED_business_telephony.html`
3. Вставить в поле "Content" в Strapi
4. Сохранить

### Шаг 3: Проверка результата

1. Обновить страницу в браузере
2. Проверить, что контент отображается
3. Проверить структуру через DevTools

## Правильная структура для business/telephony

См. файл `FIXED_business_telephony.html` - там правильная структура согласно типизации.

### Основные исправления:

1. ✅ **Удален hero-content** - он не должен быть в основном контенте
2. ✅ **Секции имеют класс `section`** - вместо `service`
3. ✅ **Все секции имеют `.container`** - обязательный элемент
4. ✅ **Контент разделен на логические секции:**
   - Секция с услугами (карточки)
   - Секция с преимуществами (grid)
   - Секция с формой заказа (service-order)
5. ✅ **Форма заказа имеет `id="order-form"`** - обязательный атрибут

## Чек-лист для каждой страницы

- [ ] Hero content удален из основного контента
- [ ] Все секции имеют класс `section`
- [ ] Специальные секции имеют правильные классы `service-*`
- [ ] Форма заказа имеет `id="order-form"`
- [ ] Все секции содержат `.container`
- [ ] Заголовки секций имеют класс `section-title` или `h2`
- [ ] Контент разделен на логические секции
- [ ] Нет дублирующихся секций

## Восстановление при ошибках

Если что-то пошло не так:

```bash
node mgts-backend/scripts/restore-from-backup.js latest

# Или восстановить только одну страницу
node mgts-backend/scripts/restore-from-backup.js latest business/telephony
```
