# Инструкция по очистке HTML контента главной страницы от дублирования

## Проблема
В HTML контенте главной страницы из Strapi присутствует дублирование элементов:
- Дублируются блоки `.about-company`
- Дублируются SVG логотипы
- Дублируются текстовые блоки и кнопки

## Решение

### Вариант 1: Через Strapi Admin Panel (Рекомендуется)

1. **Откройте Strapi Admin Panel**
   - Перейдите на http://localhost:1337/admin
   - Войдите в систему

2. **Найдите страницу главной**
   - Перейдите в **Content Manager** → **Pages**
   - Найдите страницу с slug `main_page` или названием "Главная страница"

3. **Скопируйте HTML контент**
   - Откройте страницу для редактирования
   - Найдите поле с HTML контентом (обычно это поле `content` или `content.content`)
   - Скопируйте весь HTML код

4. **Очистите HTML**
   - Запустите скрипт очистки:
     ```bash
     cd mgts-backend
     node -e "const {cleanDuplicateHTML} = require('./scripts/clean-main-page-direct.js'); const fs = require('fs'); const html = fs.readFileSync(0, 'utf-8'); console.log(cleanDuplicateHTML(html));" < ваш_файл.html
     ```
   - Или используйте интерактивный режим (см. ниже)

5. **Вставьте очищенный HTML обратно**
   - Вставьте очищенный HTML в поле контента в Strapi
   - Сохраните изменения

### Вариант 2: Интерактивная очистка

1. **Создайте временный файл с HTML**
   ```bash
   # Скопируйте HTML из Strapi в файл
   nano temp_html.html
   # Вставьте HTML, сохраните (Ctrl+O, Enter, Ctrl+X)
   ```

2. **Запустите скрипт очистки**
   ```bash
   cd mgts-backend
   node -e "
     const {cleanDuplicateHTML} = require('./scripts/clean-main-page-direct.js');
     const fs = require('fs');
     const html = fs.readFileSync('temp_html.html', 'utf-8');
     const cleaned = cleanDuplicateHTML(html);
     fs.writeFileSync('temp_html_cleaned.html', cleaned);
     console.log('✅ HTML очищен! Результат в temp_html_cleaned.html');
   "
   ```

3. **Скопируйте очищенный HTML**
   - Откройте `temp_html_cleaned.html`
   - Скопируйте содержимое
   - Вставьте в Strapi

### Вариант 3: Автоматическая очистка через API (Требует настройки)

Если у вас настроен доступ к Strapi API:

```bash
cd mgts-backend
node scripts/clean-main-page-html.js
```

**Примечание:** Этот вариант требует правильной настройки Strapi и может не работать в некоторых окружениях.

## Что делает скрипт очистки

1. **Удаляет дублирующиеся блоки `.about-company`**
   - Оставляет только первый блок
   - Удаляет все последующие

2. **Удаляет дублирующиеся SVG логотипы**
   - Оставляет только те, что находятся внутри `.about-company__content-logo`
   - Удаляет "сиротские" SVG элементы

3. **Удаляет дублирующиеся текстовые блоки**
   - Удаляет повторяющиеся заголовки и описания
   - Сохраняет только уникальные элементы

4. **Удаляет дублирующиеся кнопки**
   - Удаляет повторяющиеся кнопки "Подробнее о компании"

5. **Удаляет пустые контейнеры**
   - Удаляет контейнеры без содержимого

## Проверка результата

После очистки проверьте:
- ✅ В HTML остался только один блок `.about-company`
- ✅ Нет дублирующихся SVG логотипов
- ✅ Нет дублирующихся текстовых блоков
- ✅ Контент отображается корректно на странице

## Откат изменений

Если что-то пошло не так:
1. В Strapi Admin Panel найдите страницу
2. В истории изменений восстановите предыдущую версию
3. Или вручную верните оригинальный HTML
