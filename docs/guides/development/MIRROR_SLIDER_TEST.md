# Руководство по проверке Mirror Slider

## Как проверить, что Mirror Slider работает корректно

### 1. Проверка в консоли браузера

Откройте консоль разработчика (F12) и проверьте:

#### При загрузке страницы:
- Нет ошибок JavaScript
- Появилось сообщение о инициализации слайдера (если добавлено логирование)

#### Проверка элементов в DOM:
```javascript
// Проверить наличие слайдера
document.querySelector('.mirror-slider')

// Проверить наличие навигации
document.querySelector('.mirror-slider__navigation')

// Проверить наличие стрелок
document.querySelectorAll('.mirror-slider__arrow')

// Проверить наличие точек
document.querySelectorAll('.mirror-slider__dot')
```

### 2. Визуальная проверка

#### На главной странице:
1. ✅ Отображается секция "Почему МГТС?"
2. ✅ Видны 3 карточки слева (текстовые блоки)
3. ✅ Видны 3 изображения справа
4. ✅ Одна карточка активна (синий фон, белый текст)
5. ✅ Одно изображение активно (полная непрозрачность)

#### Навигация:
1. ✅ Видны две стрелки (влево/вправо) под слайдером
2. ✅ Видны 3 точки навигации между стрелками
3. ✅ Одна точка активна (заполнена синим)

### 3. Функциональная проверка

#### Клик по стрелкам:
- ✅ Клик на стрелку "вправо" → переключает на следующий слайд
- ✅ Клик на стрелку "влево" → переключает на предыдущий слайд
- ✅ При достижении последнего слайда, переход на первый (циклично)
- ✅ При первом слайде, переход на последний (циклично)

#### Клик по точкам:
- ✅ Клик на точку → переключает на соответствующий слайд
- ✅ Активная точка визуально выделена

#### Автоматическая прокрутка:
- ✅ Слайды автоматически переключаются каждые 5 секунд
- ✅ При наведении мыши на слайдер → автопрокрутка останавливается
- ✅ При уходе мыши → автопрокрутка возобновляется

#### Клавиатура:
- ✅ Стрелка влево → предыдущий слайд
- ✅ Стрелка вправо → следующий слайд

### 4. Проверка в разных браузерах

- ✅ Chrome/Edge
- ✅ Firefox
- ✅ Safari
- ✅ Мобильные браузеры

### 5. Проверка адаптивности

#### На мобильных устройствах:
- ✅ Слайдер перестраивается в одну колонку
- ✅ Изображение отображается сверху
- ✅ Навигация остается доступной
- ✅ Стрелки и точки имеют достаточный размер для касания

### 6. Проверка доступности

- ✅ Навигация работает с клавиатуры
- ✅ ARIA атрибуты присутствуют
- ✅ Фокус виден на интерактивных элементах

## Возможные проблемы и решения

### Проблема: Слайдер не отображается
**Решение:**
1. Проверить, что контент загружен из CMS
2. Проверить консоль на ошибки
3. Проверить, что скрипт `mirror-slider.js` подключен

### Проблема: Навигация не работает
**Решение:**
1. Проверить, что скрипт загрузился после контента из CMS
2. Проверить консоль на ошибки JavaScript
3. Проверить, что элементы `.mirror-slider` существуют в DOM

### Проблема: Автопрокрутка не работает
**Решение:**
1. Проверить, что мышь не наведена на слайдер
2. Проверить, что страница в фокусе (не в фоновой вкладке)
3. Проверить консоль на ошибки

## Команды для отладки

```javascript
// Проверить экземпляры слайдеров
window.mirrorSliders

// Принудительно инициализировать слайдеры
if (window.initMirrorSliders) {
    window.initMirrorSliders();
}

// Проверить текущий слайд
const slider = document.querySelector('.mirror-slider');
const activeCard = slider.querySelector('.open-left-slide');
const activeIndex = Array.from(slider.querySelectorAll('.mirror-slider__left-card-box')).indexOf(activeCard);
console.log('Текущий слайд:', activeIndex);
```
