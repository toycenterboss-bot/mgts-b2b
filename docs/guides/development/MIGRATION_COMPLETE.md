## Как это работает

1. **Автоматическая загрузка:** При загрузке страницы скрипт `components-loader.js` автоматически находит все элементы с атрибутом `data-component` и загружает соответствующие компоненты.

2. **Автоматическое обновление путей:** Компоненты автоматически обновляют относительные пути в зависимости от уровня вложенности страницы.

3. **Инициализация:** После загрузки компонентов автоматически инициализируется навигация и боковое меню.

## Преимущества новой архитектуры

1. **Единая точка обновления:** Изменения в header/footer нужно делать только в одном месте
2. **Меньше дублирования:** Код не повторяется на каждой странице
3. **Легче поддерживать:** Обновление компонентов автоматически применяется ко всем страницам
4. **Гибкость:** Можно легко добавлять новые компоненты
