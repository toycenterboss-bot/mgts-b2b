## 📋 Типы страниц в проекте

### 1. Главная страница (`index.html`)
- Уникальная структура с кастомным контентом из Strapi
- Секции: About Company, Почему МГТС, Наши услуги, Партнерство

### 2. Hub-страницы сегментов (9 страниц)
- `business/index.html`, `operators/index.html`, `developers/index.html`, `government/index.html`, `partners/index.html`
- Структура: Header → Breadcrumbs → Hero → Content → Footer

### 3. Категории услуг (5 страниц)
- `business/internet/index.html`, `business/telephony/index.html`, `business/security/index.html`, `business/cloud/index.html`, `business/tv/index.html`
- Структура: Header → Breadcrumbs → Hero → Content → Footer

### 4. Страницы услуг (21 страница)
- `business/internet/gpon/index.html`, `business/telephony/fixed/index.html`, и т.д.
- Структура: Header → Breadcrumbs → Hero → Content → Footer
- **Проблема:** Нет специфичных элементов для услуг (тарифы, формы заказа, FAQ)

### 5. Страницы раздела "О компании" (15 страниц)
- `about/index.html`, `about/values/index.html`, `about/ethics/index.html`, `about/governance/*`
- Структура: Header → Breadcrumbs → Hero → Sidebar + Content → Footer
- **Особенность:** Есть боковое меню (sidebar)

### 6. Контакты (`contacts/index.html`)
- Структура: Header → Breadcrumbs → Hero → Content → Footer

- `service-card` - карточка услуги (из style.css)
- `form-group` - группа формы (из style.css)

**Проблемы:**
- Нет единой системы классов
- Классы из Strapi не соответствуют базовым классам проекта
- `grid-item` используется, но не определен в CSS
- Дублирование функциональности между классами
