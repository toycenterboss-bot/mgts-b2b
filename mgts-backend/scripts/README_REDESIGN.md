# Скрипты для анализа страниц для редизайна

## analyze-page-interactive.js

Интерактивный анализ страниц с сайта `https://business.mgts.ru` с использованием Puppeteer.

### Использование

```bash
cd mgts-backend
node scripts/analyze-page-interactive.js <slug>
```

### Примеры

```bash
# Анализ страницы "О МГТС"
node scripts/analyze-page-interactive.js about_mgts

# Анализ страницы "Корпоративные документы"
node scripts/analyze-page-interactive.js corporate_documents

# Анализ страницы "Виртуальная АТС"
node scripts/analyze-page-interactive.js virtual_ate
```

### Что делает скрипт

1. ✅ Открывает страницу в браузере (Puppeteer)
2. ✅ Извлекает метаданные (title, description, H1)
3. ✅ Анализирует структуру секций
4. ✅ Извлекает изображения, файлы, ссылки
5. ✅ **Прокликивает все табы** и извлекает их контент
6. ✅ **Открывает все аккордеоны** и извлекает их контент
7. ✅ Делает скриншот полной страницы
8. ✅ Создает детальное ТЗ в JSON формате

### Результаты

Скрипт создает в `temp/page-analysis-interactive/`:

- `<slug>_spec.json` - техническое задание
- `<slug>_screenshot.png` - скриншот страницы

### Структура ТЗ

ТЗ включает:
- Метаданные страницы
- Структуру секций
- Весь контент (текст, изображения, файлы, ссылки)
- **Полный контент всех табов** (после прокликивания)
- **Полный контент всех аккордеонов** (после открытия)
- Рекомендации по компонентам

### Следующие шаги

После анализа:
1. Изучите созданное ТЗ (`<slug>_spec.json`)
2. Сверстайте новую страницу используя компоненты из `SiteMGTS/components/`
3. Используйте контент из ТЗ
4. Перенесите готовую страницу в Strapi
